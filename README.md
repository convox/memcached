# convox/memcached

Memcached docker image based on Alpine Linux. This image works with Convox Gen 2 applications.

It is similar to the
[official memcached build](https://hub.docker.com/_/memcached/) 



# Build

```bash
$ make build
```

# Run
```bash
$ make run
```
